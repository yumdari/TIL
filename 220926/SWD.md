Serial Wire Debug

SWD는 시리얼 방식으로 통신하기 때문에 클럭(SWCLK)과 데이터(SWDIO) 2개의 핀만을 사용합니다.

마찬가지로 핀에 대해서 간략하게 설명드리겠습니다.

SWDIO : Data I/O Pin.

SWO Opational trace output pin

SWCLK : Clock Pin

VCC

GND

RESET
